# ERP Sales Deck Narrative Plan

## Audience

- عميل تجاري أو صناعي صغير إلى متوسط
- مدير شركة أو شريك تشغيل
- محاسب أو مراجع يريد التأكد من الترابط المحاسبي

## Objective

تقديم النظام كنسخة ERP عربية عملية، مترابطة محاسبيًا، وجاهزة للعرض والبيع.

## Narrative Arc

1. تعريف سريع بالنظام وقيمته
2. ما الذي يديره داخل الشركة
3. الموديولات الرئيسية
4. كيف تتحرك الدورة المستندية حتى تصل إلى الحسابات
5. أين تظهر قوة الرقابة والمراجعة
6. لماذا النسخة الحالية جاهزة للعرض الآن

## Slide List

1. غلاف: تعريف بالنظام
2. ماذا يقدم النظام للشركة
3. الموديولات الأساسية
4. دورة التشغيل من المستند إلى الحسابات
5. الرقابة والتقارير والطباعة
6. حالة النسخة الحالية وجاهزيتها

## Source Plan

- README الحالي
- SALES_READY_SUMMARY
- QA_STATUS_REPORT
- لقطات شاشة حقيقية من الديمو الحالي

## Visual System

- خلفيات مأخوذة من واجهات النظام الفعلية
- طبقات بيضاء شفافة لزيادة وضوح النص
- أخضر رئيسي قريب من هوية الواجهة الحالية
- نص عربي واضح قابل للتحرير

## Asset Plan

- slide-01: dashboard
- slide-02: sales
- slide-03: purchases
- slide-04: journal
- slide-05: trial balance
- slide-06: dashboard

## Editability Plan

- كل العناوين والنصوص والبطاقات editable داخل PowerPoint
- الصور تستخدم كخلفيات سياقية فقط
- الأرقام والنقاط الرئيسية محفوظة كنصوص قابلة للتعديل
