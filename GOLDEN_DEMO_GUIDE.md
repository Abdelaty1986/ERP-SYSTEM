# Golden Demo Guide

## الشركة
- شركة جولدن ديمو للصناعات

## رحلة العرض المقترحة
1. افتح لوحة التحكم ثم العملاء والموردين والأصناف.
2. اعرض أمر الشراء ثم إذن الإضافة ثم فاتورة المورد.
3. اعرض أمر البيع ثم إذن الصرف ثم فاتورة البيع.
4. اعرض سند القبض وسند الصرف.
5. انتقل إلى كشف حساب العميل وكشف حساب المورد.
6. راجع اليومية ثم الأستاذ ثم ميزان المراجعة.

## البيانات الأساسية
- العميل الخاضع: `ID 1`
- العميل العادي: `ID 2`
- المورد الخاضع: `ID 1`
- المورد المعفي: `ID 2`

## المشتريات
- أمر شراء: `ID 1` -> `/purchase-orders`
- إذن إضافة: `ID 1` -> `/purchase-receipts`
- فاتورة مورد خاضعة: `ID 1` -> `/purchases`
- فاتورة مورد معفاة: `ID 2` -> `/purchases`

## المبيعات
- أمر بيع: `ID 1` -> `/sales-orders`
- إذن صرف: `ID 1` -> `/sales-deliveries`
- فاتورة بيع مخزنية: `ID 1` -> `/sales`
- فاتورة بيع مالية: `ID 1` -> `/sales/financial`

## الخزينة
- سند قبض: `ID 1` -> `/receipts`
- سند صرف: `ID 1` -> `/payments`

## روابط طباعة مباشرة
- فاتورة البيع: `/sales/1/print`
- فاتورة المورد: `/purchases/1/print`
- أمر الشراء: `/purchase-orders/1/print`
- سند القبض: `/receipts/1/print`
- سند الصرف: `/payments/1/print`

## تقارير المراجعة
- كشف حساب العميل الخاضع: `/customers/1/statement`
- كشف حساب المورد الخاضع: `/suppliers/1/statement`
- اليومية: `/journal`
- الأستاذ النقدي: `/ledger/1`
- ميزان المراجعة: `/trial-balance`
- تقرير العملاء: `/reports/customers`
- تقرير الموردين: `/reports/suppliers`
- تقرير المخزون: `/reports/inventory`
